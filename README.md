# Python Basics Projects

Beginner Python projects, part of an 8-week self-guided roadmap to build automation and cybersecurity skills.

## Projects
- `calculator.py` - basic calculator supporting +, -, *, /
- `temp_converter.py` - converts Celsius to Fahrenheit
- `shopping_list.py` - collects grocery items into a list
- `expense_tracker.py` - tracks expenses and calculates total spend

More projects added weekly as the roadmap progresses.
